## Informações Gerais
API em NodeJS que consiste em criar uma calculadora de números romanos, registro e autenticação de usuários
Feita usando Clean Architeture e DDD respeitando os princípios SOLID

* O Usuário pode fazer um cadastro comum com: nome, email e senha
* O Usuário pode fazer login ou logout
